Q1: What is the main purpose of memory in an AI agent?

A. To increase internet speed
B. To remember and use relevant information over time
C. To reduce screen brightness
D. To replace databases

Answer: B

Q2: Which type of memory is mainly used to handle the current conversation?

A. Long-term memory
B. Persistent memory
C. Short-term memory
D. External memory

Answer: C

Q3: Long-term memory is mainly used for:

A. Managing only the current message
B. Storing information across multiple interactions
C. Deleting user context
D. Reducing model size

Answer: B

Q4: Which statement is correct?

A. Short-term memory is always permanent
B. Long-term memory is usually session-only
C. Short-term memory helps with immediate context
D. Long-term memory is never useful

Answer: C

Q5: A stateless AI agent is one that:

A. Has unlimited memory
B. Treats every interaction independently
C. Remembers every past conversation
D. Uses only long-term memory

Answer: B



Q6: A learning assistant remembers that a student prefers simple examples. This is an example of:

A. Short-term memory only
B. Persistent memory
C. Data cleaning
D. Message truncation

Answer: B

Q7: Which of the following is a poor candidate for persistent memory?

A. User prefers Python examples
B. User is preparing for interviews
C. User said “hello” once
D. User likes concise explanations

Answer: C

Q8: Episodic memory in AI is most similar to:

A. A list of mathematical formulas
B. A record of past events or interactions
C. A random password generator
D. A deleted message log

Answer: B
