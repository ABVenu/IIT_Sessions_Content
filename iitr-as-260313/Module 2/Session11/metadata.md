Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 2hr 15 mins

title: Self-Reflection and Feedback Loops

objective: Understand how agents improve responses through reflection and feedback	

topics be covered: self-correction prompts;iterative prompting;agent prompt design


detailed subtopics to be covered:

- Define Self-Reflection — Explain how AI agents review their own responses, why reflection improves accuracy, and introduce the idea of “thinking twice” before final answers;
- Introduce Self-Correction Prompts — Show how to design prompts that ask the model to verify, critique, and fix its own output with simple before–after examples;
- Apply Iterative Prompting — Demonstrate how to improve responses step-by-step through multiple prompt cycles instead of expecting perfect output in one go;
- Design Feedback Loops — Explain how agents use feedback (user input or internal checks) to refine responses continuously;
- Build Agent Prompts — Teach how to combine instructions, reflection steps, and feedback into a single prompt flow for smarter agent behavior;
- Compare Improvement Strategies — Differentiate one-shot prompting vs iterative prompting vs reflection-based prompting with practical use cases;
- Practice Prompt Refinement — Engage learners in improving a weak response using self-correction and iterative prompting techniques;
- Evaluate Response Quality — Identify errors, gaps, and hallucinations, and learn how to guide the model toward better outputs;
- Optimize Feedback Loops — Share best practices to design efficient, reusable reflection workflows for real-world AI agents;
