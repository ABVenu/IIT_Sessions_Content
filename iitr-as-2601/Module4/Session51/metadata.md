Course Name: Certification in Agentic Systems and Design

Target Audience : Students from any backgorund may not be necessarily form tech background


session duration: 1hr  50mins

title: Multimodal Pipelines (Speech and Vision Models)



objective: Build a speech pipeline that transcribes audio, summarizes the text, speaks the summary, and try a lite vision step.

type of session: theory/implementation/mixture of theory + implementation / You take the call

topics be covered: Speech-to-Text; Transcript summarization; Text-to-Speech integration; Vision basics


detailed subtopics to be covered:
- Map a multimodal agent pipeline: audio in → text → summary → audio out, and where a simple image → description step fits.
- Run Speech-to-Text on a short audio sample and check transcript quality.
- Summarize the transcript with a known cloud model (Groq in notes; Ollama Cloud as the other option to try) into a short useful summary.
- Integrate Text-to-Speech for the summary, run full Speech-to-Text → summary → Text-to-Speech once, and try a lite vision step (image → short description).
