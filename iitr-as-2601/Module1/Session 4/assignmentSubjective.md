## Subjective Assignment: Python Fundamentals for AI

**Total: 3 Questions | Format: Coding / Implementation**

---

### 2.1 Easy — User Age Calculator

**Task:**
Write a Python program that asks the user for their birth year and prints their age assuming the current year is 2025.

**Constraints:**

* Use `input()`
* Convert types explicitly

**Sample Input:**
`2000`
**Expected Output:**
`You are 25 years old.`

---

### 2.2 Intermediate — Simple Accuracy Reporter

**Task:**
Ask the user to enter a floating-point accuracy value. Print it using an f-string with a descriptive message.

**Sample Input:**
`0.91`
**Expected Output:**
`Model accuracy is 0.91`

**Stretch:** Validate that input is numeric.


---

### 2.3 Hard — Interactive Profile Builder

**Task:**
Build a small program that:

1. Asks for a user’s name
2. Asks for their age
3. Asks whether they are an active user (`True`/`False`)
4. Prints a clean, formatted summary

**Requirements:**

* Explicit type conversions
* Meaningful variable names
* Use at least one f-string

**Sample Output:**
`User Alice is 25 years old. Active status: True`
