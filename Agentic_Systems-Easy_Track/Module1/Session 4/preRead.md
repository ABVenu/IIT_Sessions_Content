## Introduction: Why Python?
Think of Python like a common language spoken by people from many countries.

Different tools, libraries, and platforms all “agree” to use Python because it is:
- Easy to read  
- Easy to write  
- Powerful without being complicated  

That’s why AI engineers, researchers, and beginners all start here.

## Why Learning Python Matters
Before AI models and libraries, there is **plain Python code**.

Learning Python helps you:
1. **Understand what the computer is doing**  
2. **Control data** instead of guessing  
3. **Read and modify other people’s code**  

Without these basics, AI tools feel like magic. With them, they feel logical.

## From Known to New: Values → Variables
### The Way You Already Think
In real life, you label things:
- Your name  
- Your age  
- Your phone number  

You don’t repeat the value every time—you use the label.

### The Python Way
A **variable** is a label that stores a value.

Instead of repeating numbers or text, you store them once and reuse them.

---

## Core Python Building Blocks
### 1. Variables
Variables store information so you can use it later.

- They have a **name**  
- They hold a **value**  
- The value can change  

*Think of a variable as a named box.*

---

### 2. Data Types
Python understands different kinds of data.

- **Int** → whole numbers (10, -3)  
- **Float** → numbers with decimals (3.14, 0.5)  
- **String** → text ("hello")  
- **Bool** → true or false values  

Each type behaves differently.

---

### 3. Basic Math in Python
Python can do everyday math:
- Addition  
- Subtraction  
- Multiplication  
- Division  

Math operations work directly on numbers stored in variables.

---

### 4. Print and Input
- **print()** shows information on the screen  
- **input()** lets the user provide information  

Together, they allow simple interaction between you and the program.

---
