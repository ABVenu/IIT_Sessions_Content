### Question: Modular Student Evaluation System

**Task:**
Build a small, modular Python program that evaluates a student’s performance using functions.

Your program must include **at least the following functions**:

1. A function that takes a student’s name as input and returns a formatted greeting.
2. A function that takes a list of numeric scores and returns:

   * the number of subjects
   * the average score
3. A function that takes the average score and returns:

   * `"Pass"` if the average is 50 or above
   * `"Fail"` otherwise
4. A `main()` function that:

   * Calls all other functions
   * Prints the final result in a clean, readable format

**Requirements:**

* Use parameters (do not hardcode values inside functions)
* Use `return` (not `print`) for computed results
* Use multiple functions with clear responsibilities
* Call `main()` to start the program

**Sample Output (example):**

```
Hello, Alice
Subjects: 3
Average Score: 65.0
Result: Pass
```

This question is designed to test:

* Function definition and calling
* Parameters vs arguments
* Returning values
* Multiple return values
* Modularity and program structure
* Realistic program flow
