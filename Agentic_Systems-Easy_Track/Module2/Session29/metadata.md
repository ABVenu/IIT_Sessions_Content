title: Regression: Regularization and Evaluation

objective: Understand how to control model complexity and evaluate regression performance

topics to be covered: 
- Regularization – intuition behind Ridge and Lasso regression; 
- Model Comparison – impact of regularization on model behavior; 
- Evaluation Metrics – MAE RMSE and R-squared interpretation; 
- Error Analysis – identifying patterns in prediction errors; Practical Perspective – choosing metrics and evaluating model performance
