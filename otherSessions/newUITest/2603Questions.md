# 2603 Objective Questions (Module 1)

1. Which function is used in Python to take user input from the keyboard?  
   A. `print()`  
   B. `input()`  
   C. `read()`  
   D. `scan()`  
   **Answer:** B

2. Which conditional block runs when all previous conditions are false?  
   A. `if`  
   B. `for`  
   C. `else`  
   D. `while`  
   **Answer:** C

3. Which loop is best suited when the number of iterations is known in advance?  
   A. `while`  
   B. `for`  
   C. `if`  
   D. `def`  
   **Answer:** B

4. What is the correct way to access the first element of a Python list `nums`?  
   A. `nums(0)`  
   B. `nums[1]`  
   C. `nums[0]`  
   D. `nums.first()`  
   **Answer:** C

5. Which Git command records staged changes to the local repository history?  
   A. `git push`  
   B. `git commit`  
   C. `git clone`  
   D. `git pull`  
   **Answer:** B

6. In NumPy, which property is used to check the shape of an array?  
   A. `array.size`  
   B. `array.shape`  
   C. `array.type`  
   D. `array.index`  
   **Answer:** B

7. Which SQL clause is used to filter rows based on a condition?  
   A. `GROUP BY`  
   B. `ORDER BY`  
   C. `WHERE`  
   D. `LIMIT`  
   **Answer:** C

8. In Pandas, which function is commonly used to read a CSV file into a DataFrame?  
   A. `pd.load_csv()`  
   B. `pd.read_csv()`  
   C. `pd.open_csv()`  
   D. `pd.import_csv()`  
   **Answer:** B

9. Which Python data type stores key-value pairs?  
   A. List  
   B. Tuple  
   C. Dictionary  
   D. Set  
   **Answer:** C

10. Which SQL keyword removes duplicate values from query results?  
    A. `UNIQUE`  
    B. `DISTINCT`  
    C. `FILTER`  
    D. `GROUP`  
    **Answer:** B
